### Getting started

Read the [mkdocs-material guide](https://squidfunk.github.io/mkdocs-material/getting-started/).

### How to serve local website

Run:

```bash
mkdocs serve
```

### How to deploy to GitHub

Run:

```bash
mkdocs gh-deploy
```

### How to add a Topic

```
node add-topic.js
```

node add-page.js
